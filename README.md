# Crackle Media

Crackle Media is a online movie and tv series exploration site with features of auth using (TMDB) and next auth and leverages NextJS latest features like app router.

## Techstack
- NextJS 13
- Next Auth
- TMDB api
- React
- Tailwind CSS
- Vercel(Deployment)
- Railway(Deployment)
- Docker
